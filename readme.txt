Instalación:
      
1. copiar en el servidor el directorio TDA.

2. colocar el user y password del mysql del servidor en 
   el archivo includeLibs.php que se encuentra en el
   siguiente directorio TDA/lib/includeLibs.php

3. crear la base de datos en mysql con el nombre de TDA

4. cagar el script install.sql en la base de datos que 
   se encuentra en el directorio TDA/sql/install.sql
